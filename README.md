# GetItDone
A website with intelligent chatbot that suggests best doctors and lawyers based on your requirements.

## Technologies Used
MERN stack is used to build the website.

This project uses docker containers(linux).

## Installation
### Prerequisites
Make sure docker and docker-compose is installed in linux.
If you are using Docker Desktop in Windows make sure your are using linux containers (Hyper-V or WSL2).

Use `git clone` to clone the repository.
> git clone https://github.com/gopaluniphani/get-it-done

To run the project
> cd get-it-done

> docker-compose up -d

**Now go to localhost:8080 in your favorite browser**
